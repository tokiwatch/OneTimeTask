OneTimeTaskプラグイン (プラグインの雛形)
===================================

はじめに
--------

このプラグインは、「1日1回何かする」プラグインを制作する際の雛形です。

インストール
------------

1. (事前準備) run-periodic-taskが定期的に実行されるようにcronを設定してください。
1. プラグインを、MTのプラグインディレクトリに設置してください。
